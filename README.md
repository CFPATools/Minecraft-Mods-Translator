# Minecraft Mods Translator

适用于Minecraft 1.16+版本的MC模组翻译器

![](README/readme-1.png)

## 使用方法

1.双击打开模组翻译器

2.选中MOD的en_us.json打开

## 软件功能

格式化字符智能提示与比对
